# Fleska task assignment 

## Credential of my task 

- Technology (node js)
- Libs (express, cors, mongoose, nodemon, dotenv)
- Tools (Vs code, Chrome) 

## Installation

- Clone repo from github or download zip file
- install the packages 
```sh
npm install
npm run dev
(or)
npm start
after that open  127.0.0.1:8000
```
